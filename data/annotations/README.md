# PEDB Annotation Files

## Canonical (v2.0 final)
- `pedb_sentence_annotations.csv`: sentence-level 5-label annotations.
- `pedb_explanation_ratings.csv`: explanation-level ratings used in paper aggregates.

## Legacy (archived)
- `pedb_annotations_master.csv`
- `pedb_annotations_overlap.csv`

Legacy files were early 10-move templates and are retained for traceability only.
